歡迎使用 MCLang API

這是一個專為 MCLang 設計的現代化 Web API 介面。您可以在此取得所有的 Minecraft 翻譯檔案與字典，並進行自動化的文字建置。

📖 如何使用？

🔑

1. 設定連線與金鑰

前往左側「連線與認證設定」，確認您的 API 伺服器位址是否正確。若伺服器開啟了安全認證，請向管理員索取 API Key 並填入。

📄

2. 查閱 API 支援

API 提供 語言清單查詢 與 單字翻譯解析 功能。支援多版本 Minecraft 官方翻譯語言包。

🧪

3. 進行測試

在下方的「API 測試區」輸入參數，即可動態取得回應，無需編寫程式碼即可預覽格式與內容。

連線與認證設定

定義後台 API 的基準網址 (Base URL)。若有配置 requireKey 安全機制，請在此設定金鑰。

API 測試區 (Playground)

選擇下方的端點 (Endpoint) 輸入對應的參數進行互動發送。若是尚未授權的請求將會遭到伺服器攔截。

GET /api/languages

取得目前伺服器支援的所有語言與版本清單。

GET /api/languages/{lang}

取得特定語言的翻譯列表。

GET /api/translate?lang={lang}&key={key}

取得特定語言的單一翻譯字串。
