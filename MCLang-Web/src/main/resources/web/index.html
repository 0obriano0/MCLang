MCLang API

Welcome to MCLang API

Minecraft Library API is a web-based portal to access Minecraft translation files, allowing you to fetch, test, and automatically build localized components for your development community.

How to Use?

1. Setup Connection

Setup your connection rules to the API server within the Auth & Connection settings.

2. Check Supported APIs

Check supported APIs for language lists and dictionary references.

3. Run Tests

Run tests and preview API responses dynamically within the playground.

Auth & Connection Settings

Define the Base URL for the background API. If the server requires a key, please configure it below.

API Testing Playground

Select an endpoint below to send a request to the server. Unauthorized requests will be blocked if security is enabled.

Developer Java API Guide

How to use MCLang as a dependency in your Bukkit plugin.

1. Maven Dependency

<repositories>
  <repository>
    <id>maven</id>
    <url>https://repo1.maven.org/maven2/</url>
  </repository>
</repositories>

<dependency>
  <groupId>org.tsob</groupId>
  <artifactId>MCLang-API</artifactId>
  <version>1.2.2</version>
  <scope>provided</scope>
</dependency>

2. Usage Example

import org.tsob.MCLang.API.MCLang;

public class ExamplePlugin extends JavaPlugin {
  @Override
  public void onEnable() {
    MCLang lang = new MCLang("en");
    ItemStack item = new ItemStack(Material.DIAMOND_SWORD);
    String translation = lang.getItemTranslate(item);
  }
}
